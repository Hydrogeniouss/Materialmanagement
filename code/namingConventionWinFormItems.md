# Naming Convention for Windows Forms Items

| Symbol | Item        |
| ------ | ----------- |
| lbl    | Label       |
| btn    | Button      |
| rad    | Radiobutton |
| cbx    | Combobox    |
| xbx    | Checkbox    |
| gbx    | Groupbox    |
| tbx    | Textbox     |
| lbx    | Listbox     |
| sbx    | Splitbox    |
| tmr    | Timer       |
| dtp    | Date        |
